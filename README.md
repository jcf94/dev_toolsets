# Dev Toolsets

Some dev tool scripts.

## Build dev tools

Currently, this will:

- Build **GCC**
- Install **Miniconda3**
- Build **LLVM**
- Build **OneDNN** [Optional]
- Install **CUDA** [Optional]

```bash
./build.sh
```

## Activate dev environments

Activate the environment if it exists:

- **GCC**
- **Miniconda3**
- **LLVM**
- **OneDNN**
- **CUDA**

```bash
source ./env.sh
```
