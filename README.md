# Dev Toolsets

Some dev tool scripts.

## Build dev tools

Currently, this will:

- Build *gcc*
- Install *miniconda*
- Build *llvm*
- Build *OneDNN* [Optional]
- Install *CUDA* [Optional]

```bash
./build.sh
```

## Activate dev environments

Activate the environment if it exists:

- gcc
- miniconda
- llvm
- onednn
- cuda

```bash
source ./env.sh
```
